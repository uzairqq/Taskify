# Taskify – Full-stack Task Management (Monorepo)

**Stack:** ASP.NET Core Web API + React (TypeScript)

## Structure

- `backend/Taskify.Api` – Web API (Swagger)
- `frontend/taskify-web` – React + TS (Vite)
- `docs/` – notes, roadmap

## Run Backend

cd backend/Taskify.Api
dotnet run

## Run Frontend

cd frontend/taskify-web
npm install
npm run dev
